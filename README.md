# Oshimeguri Map

A fan-made map to explore anime, drama, and movie locations in Japan.

アニメ・ドラマ・映画の聖地やロケ地を地図でたどるファン向けマップサイトです。

---

## Overview

This project aims to collect and visualize real-world locations that appear in anime, dramas, and movies.

このプロジェクトは、作品に登場する実在の場所を地図上でまとめることを目的としています。

---

## Features

- View locations on Google Maps  
- Simple and lightweight static site  
- Easy to extend with JSON data  

- Google Mapsで場所を確認できる  
- シンプルな静的サイト構成  
- JSONで簡単にデータ追加可能
